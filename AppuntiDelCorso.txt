Packeto script ci permete di crearre un projetto c# semplice!
instalato Packeto globale script:
dotnet tool install -g dotnet-script

cd cartella dove vogliamo:
dotnet script init

Comando per fare il ReverseEngeneering del Db:
   dotnet ef dbcontext scaffold "Data Source=Data/MyCourse.db" Microsoft.EntityFrameworkCore.Sqlite --output-dir Models/Entities
   --context-dir Models/Services/Infrastructure --context MyCourceDbContext
